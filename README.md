# Desafios de Lógica de Programação em Java

Repositório dedicado ao estudo, treino e consolidação dos conceitos fundamentais de algoritmos e lógica de programação utilizando a linguagem Java.

---

## Conceitos Praticados

- **Estruturas de Controle e Decisão:** Condicionais (`if`, `else`), estruturas aninhadas e comparação de textos (`.equals()`, `.equalsIgnoreCase()`).
- **Estruturas de Repetição:** Laços `for` (iteração direta e regressiva) e laço `while` para menus interativos.
- **Estruturas de Dados e Vetores:** Manipulação de arrays, busca por índice e associação por vetores paralelos.
- **Manipulação de Strings:** Varredura com `charAt()`, medição com `length()`, conversão de caixa (`toUpperCase()`, `toLowerCase()`) e concatenação.
- **Operadores Matemáticos e Lógicos:** Operador de resto (`%`), contadores, acumuladores e variáveis de controle.
